UI Explanation

This is our app UI Design (our app is not live yet, currently we're working on App UI and strategies). Here is the explanation of the three UI images:

Homepage UI Design (mark as 1) :
  • In the homepage image (that mark as 1), you can see five components Header, Footer, Banner, Top Users, Ongoing Plays.
  • Header contains the hamburger, logo and search icon. Hamburger opens the slider that contains many links to easily navigate with the UI.
  • Footer contains the four links of four main pages that as follows Homepage, Leaderboard, Games, Profile.
  • Banner section contains the latest and upcoming events like Weekly test series and some sponsored banners.
  • Top Users section contains our top active, winners and top player by their performance.
  • Ongoing Plays section contains the plays that are available to play. These banners contains all information like specific field/subject, winning amount, entry fees, duration, total questions etc.
  • If the user not found their specific subject/field plays, they can also post on our platform.
Leaderboard UI Design (mark as 2) :
  • In the leaderboard UI image (that mark as 2), you can see components like Header, Top Players.
  • Top Players contains the Top players info based on This week/ This month.
Profile UI Design (mark as 3) :
  • In the profile image (that mark as 3), you can see many components like User Info, My Stats, Setting, Withdraw.
  • User Info contains the user info like user profile, image, current rank, total wallet amount, total plays, total wins, global rank.
  • My Stats contains complete statistics about user and their plays.
  • Setting section contains the user personal info and some other settings to change.
  • Withdraw section contains the payment method to withdraw money directly in their account.
